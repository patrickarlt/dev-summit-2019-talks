Vector tiles are pre-generated and then cached on a server Vector basemaps are composed of many layers that represent geographical types such as land, water, roads, buildings, and text labels
What is a style? A document that defines the visual appearance of the map - what to draw, order to draw it in, how to style. Styles are written as a large JSON object with root level and nested properties. Layers are drawn in the order that they are listed in a root-level layers property Layers except for background type refer to a source. Layers can optionally filter features Layers must have a type
